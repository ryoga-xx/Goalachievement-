# 目標達成アプリ
1. 初期設定（AIコーチとの対話）
アプリを開くと、まずAIコーチとのチャット（オンボーディング）が始まります。
AIからいくつか質問（「どんな未来を実現したいか」「現在どんなことに悩んでいるか」など）が投げかけられます。
リラックスして、あなたの素直な思いを入力してください。
すべての質問に答え終わると、AIがあなたの回答を分析し、あなた専用のダッシュボード（ホーム画面）を自動生成します。
2. ホーム画面（あなたの司令塔）
ダッシュボードが完成すると、メイン画面が表示されます。
未来への没入（FIREの聖域）： 画面上部にあなたの「理想の未来」が表示されます。中央の波打つオーブをクリックすると、10分間の瞑想タイマーが起動します。朝や作業前に行い、理想の未来を鮮明にイメージしましょう。
主要な成長の柱（目標）： AIと一緒に立てた大きな目標（最大3つ）が表示されます。各目標をクリックすると、「SMART-ER法」に基づいた具体的なプランや、「もし壁にぶつかったらどうするか（If-Thenプランニング）」の詳細を確認できます。
今日の小さな一歩（習慣）： 目標達成のために毎日行うべき「習慣」のリストです。
3. 日々の実行と「ログ」の記録（振り返り）
実際に行動を起こしたら、画面下のメニューバーから**「ログ」タブ**を開きましょう。
気分のトラッキング： 行動する「前」と「後」の気分を数値で記録します。「最初は面倒だったけど、やってみたらスッキリした」という成功体験を脳に覚えさせます。
一言リフレクション： その日の気づきを自由にメモします。
保存して祝福： 「今日のログを保存する」を押すと、紙吹雪でお祝いされます。これを毎日続けることで「連続達成（ストリーク）」がカレンダーに記録され、モチベーション維持の強力な味方になります。
4. 目標の達成とゲーミフィケーション
大きな目標を達成したら： 目標詳細画面で「目標を達成した！」ボタンを押してください。特別なセレブレーション（祝福）画面へと移行し、あなたの経験値が貯まり、レベルアップします！
目標の管理： 画面下の**「目標」タブ（🏆アイコン）**から、新しく目標を追加したり、過去に達成した目標のリスト（殿堂入り）を眺めて自信を深めることができます。
5. 緊急時のSOSコーチ
画面右下に常に表示されている**「SOS」マークの丸いボタン**は、あなたの専属コーチです。
「どうしてもやる気が出ない」「失敗して落ち込んでいる」「タスクが多すぎてパニック」という時は、遠慮なくこのボタンを押して、今の気持ちを吐き出してください。AIがあなたのメンタルを立て直すサポートをします。
